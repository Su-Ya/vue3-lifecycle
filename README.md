# Vue 3 Lifecycle Demo

Observe the Vue 3 lifecycle hooks of the **Composition API** and **Options API** with a custom **Console Viewer**.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open your browser and navigate to the local server URL (typically `http://localhost:5173`) to interact with the lifecycle playground!
